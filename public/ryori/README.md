# Ryori - QR Code Ordering & POS System

Welcome to the Ryori project showcase! This folder contains screenshots and documentation for the innovative restaurant ordering and point-of-sale system developed at ALTTOS Innovations Corporations.

## Project Overview

**Ryori** is a nationwide QR code ordering and POS system designed to revolutionize the dining experience for restaurants and their customers. The application streamlines operations, simplifies the ordering process, and empowers entrepreneurs in the food and beverage industry.

## Project Details

- **Company**: ALTTOS Innovations Corporations
- **Role**: Software Developer Intern
- **Status**: Currently Active
- **Location**: Davao City, Philippines
- **Type**: Cross-Platform Mobile Application

## Key Responsibilities

- Developing a cross-platform mobile application using React Native
- Implementing efficient local data storage solutions (MMKV & Watermelon DB)
- Collaborating with backend teams to integrate MongoDB databases
- Ensuring seamless data flow and synchronization
